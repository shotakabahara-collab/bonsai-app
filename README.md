# BONSAI

**育てた時間が、作品になる。**

iPhoneのホーム画面から遊べる、独立型の盆栽育成PWAです。

## 実装済み

- 黒松・山もみじ・皐月の選択
- 現実時間の10倍速育成
- 水やり・針金・施肥・鉢替え
- 確定後に取り消せない剪定
- 師匠チュートリアル
- 顔付きの師匠・ライバル・審査員・評論家・富豪
- 非同期型の週次大会
- 盆栽師称号、受賞歴、生涯年表、銘木録
- 鑑賞モード、端末保存、オフライン起動

公開URL: https://shotakabahara-collab.github.io/bonsai-app/

GitHub Pagesの公開元はGitHub Actionsに設定済みです。
